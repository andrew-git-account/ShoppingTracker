HOW TO RUN THE APP:
===================

1. EASIEST WAY (Windows):

   Double-click this file: run_app.bat

   That's it!


2. What you'll see:

   The app will start and show:
   [OK] Application ready!
   [OK] Running on: http://127.0.0.1:5000


3. Open your browser:

   Go to: http://localhost:5000


4. What you can do RIGHT NOW (without API key):

   - Click "History" tab
   - See 2 sample receipts (Walmart and Target)
   - Click on receipts to expand/collapse them
   - See items, prices, totals


5. To upload real receipts:

   - You need to add your Anthropic API key to .env file
   - Get key from: https://console.anthropic.com/
   - Edit .env file, add: ANTHROPIC_API_KEY=sk-ant-your-key-here
   - Restart the app


6. To stop the app:

   - Press Ctrl+C in the terminal window
   - Or just close the window
